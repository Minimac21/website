Mounting a Drive on a Linux System

sda can be sdb or mmlocm or whatever.

  1. lsblk and fdisk -l to get info
  2. If there isn't a numbered /dev/sda1 or such OR if fdisk -l shows only disks, and no "Devices" with starts & ends, it isn't partitioned, do step 3.
  3. Partitioning: fdisk is old (MBR tables), gdisk is newer and better (GPT tables). This wipes contents btw.
    • gdisk
    • n (make a new partition)
    • all defaults, can be strategic about partitions (boot, swap, home, /)
    • w (write)
  4. Testing: sudo mount /dev/sda1 /mnt
  5. If you're going to mount somewhere with existing data, right now you want to copy the contents to /mnt
  6. umount /dev/sda1
  7. mount /dev/sda1 <wherever>
  8. To make this happen automatically at boot, edit fstab
  9. Get UUID with lsblk --fs
  10. To make mounting happen on boot, add a line to etc/stab with the following format (spaces or tabs separating):
  11. UUID="<insert>" mountpoint fstype(ext4?) defaults 0 (1 if root, 2 if anything else)
